# 超簡版操作流程

## 你真正要做的只有這些

### 1. 建立 D1 資料庫

在 Cloudflare 裡建立一個 D1 database，名稱建議：

boss-schedule-db

### 2. 建立資料表

把 `schema.sql` 裡面的內容貼到 D1 Console 執行。

### 3. 上傳這包到 Cloudflare Pages

把整個資料夾上傳到 Pages。

### 4. 綁定 D1

在 Pages 專案設定裡新增 D1 Binding：

Binding name 必須填：

DB

Database 選：

boss-schedule-db

### 5. 重新部署

重新部署後，公會大家打開同一個網址，就會共用資料。

## 出問題時先看這個

如果網頁右上角出現「目前是本機測試模式」：
代表網站沒有連到資料庫，通常是 D1 Binding 沒綁好，或 Binding name 不是 DB。
